Welcome to bluetermgen's documentation!
=======================================

``bluetermgen`` is a Python library for creating terminal UI elements with borders and styling.
Easily generate messages, menus, and tables with customizable styling, alignment, and formatting.

Features
--------

* Create bordered message boxes with various styles
* Build interactive menus with customizable indexing
* Display data as formatted tables with headers and alignment
* Control alignment, padding, and styling for all elements
* Support for multiple border styles (single, double, bold, ASCII)
